[en]

Sample React application built with NextJS and Chakra UI

## How to run

- Clone the repo
- Enter the project's folder
- Install the dependencies
- Execute build
- Execute start

```bash
git clone https://github.com/leonardo015/milk-check.git
cd milk-check
yarn
yarn build
yarn start
```
## Access the app via http://localhost:3000
